# Real-Time Chat Application

A modern real-time chat application built with React, TypeScript, and Socket.IO. Features include user authentication, instant messaging, typing indicators, and online presence.

## Tech Stack

- **Frontend Framework:** React 19 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS v4
- **State Management & Data Fetching:** TanStack Query
- **Real-Time Communication:** Socket.IO Client
- **Form Handling:** React Hook Form + Zod
- **UI Components:** Radix UI
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Toast Notifications:** Sonner
- **Routing:** React Router DOM
- **Authentication:** JWT with Universal Cookie
